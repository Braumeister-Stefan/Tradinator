#   - QuantConnect Lean.Brokerages.IG / IGSymbolMapper.cs (github.com/fabiankliem/Lean.Brokerages.IG)
#   - ig-python/trading-ig sample_utils.py and test_integration.py (github.com/ig-python/trading-ig)

#   - Tradinator universe.json (existing validated/candidate epics)
#
# Format: EPIC | Instrument Name | Asset Class | Sub-Class | Region
# NOTE: All epics must be verified against the IG Demo API via discover_universe.py
#       before being used in live paper trading. Availability may vary.
#
# Epic code conventions:
#   IX.D.xxx.DAILY.IP  -> Spread-bet daily (indices & single stocks)
#   CS.D.xxx.MINI.IP   -> CFD mini (forex)
#   CS.D.xxx.TODAY.IP  -> Spread-bet rolling (forex, spot metals, crypto)
#   CS.D.xxx.CFD.IP    -> CFD (crypto, some forex)
#   CC.D.xxx.USS.IP    -> Commodity DFB rolling (energy, metals, agri)
#   CC.D.xxx.UMP.IP    -> Commodity DFB (alternative code, some markets)
#   MT.D.xxx.MonthN.IP -> Metals futures (near-dated)
#   EN.D.xxx.MonthN.IP -> Energy futures (near-dated)
#   IR.D.xxx.MonthN.IP -> Interest rate futures
#   CO.D.xxx.MonthN.IP -> Agricultural commodity futures
#   CF.D.xxx.MON.IP    -> FX futures
#   IX.D.SUN*.DAILY.IP -> Weekend spread-bets
